# FreelanceX — Freelancing Platform

A full-stack web platform that connects clients with freelancers, allowing clients to post jobs and freelancers to create profiles and find work. Built as a complete end-to-end application with a database-driven backend.

## Features

- User registration and login with role-based access (client / freelancer)
- Job posting and browsing
- Freelancer profiles
- Database-backed storage for users, jobs, and related data
- Server-side input validation

## Tech Stack

- **Backend:** PHP
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript

## How to Run

1. Install a local server environment such as [XAMPP](https://www.apachefriends.org/) (Apache + MySQL + PHP).
2. Copy the project folder into the `htdocs` directory.
3. Open phpMyAdmin and import the included database file (`database.sql`) to create the required tables.
4. Update the database connection settings in the config file if needed (host, username, password).
5. Start Apache and MySQL from the XAMPP control panel.
6. Open `http://localhost/FreelanceX` in your browser.
